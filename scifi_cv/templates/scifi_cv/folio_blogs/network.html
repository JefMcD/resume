Summary :

This project is a Social Network App I developed using CSS & SASS for the front end, with a JavaScript Client and a Python/Django back-end server and API. Its designed using a MySQL Database on Ubuntu Linux .


Developed using VSCode, Graphics created using Inkspace and GIMP, Design Documentation produced using MySQL Workbench, and DrawIO and presentation made using Blender, Reaper and Libre Office.


Tech Used:

Django,Python,CSS/SASS,JS,MySQL

Graphics Apps Used :

Krita, Gimp, Inkspace
