Hungry Hippo

Website Demo: Hungry Hippo
Guthub: Git Hub

Summary :

Hungry Hippo Image Batch Processor written in Django, Python and MySQL for the backend with a Javascript client making asynchronous fetches through the API to provide dynamic contents. The front end User Interface is written using CSS and SASS.

The App allows a folder of images to be uploaded for them to be cropped, rotated, resized and tagged in a way thats simple and intuitive. This is a task often carried out to produce content for online shops, websites.

Tech Used:

Django,Python,CSS/SASS,JS,MySQL

Graphics Apps Used :

Krita, Gimp, Inkspace, Blender
